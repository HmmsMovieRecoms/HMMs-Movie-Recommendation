The current directory contains the C codes and the experimental data used in [1].
__________________________________________________________________________________________________

The three following sub-directories are available:

1- 'ExampleArticle': Codes and data of the example presented in the Tables 2 and 7
                     of [1] where three users u, v and w watched the movies x1 to x7.
					 
2- 'HybridSystem': Experimental codes and data used in [1] for the 'MovieLens-100K' dataset.

3- 'OtherFiles': User ratings files in various formats, and files related to the frequencies 
                 of the movie genres, of the user occupations, and of the user zip codes for 
				 the 'MovieLens-100K' dataset.
__________________________________________________________________________________________________

[1] J. CHIASSEUP KEUDJEU, O. KENGNI NGANGMO, SYLVAIN ILOGA and T. BOUETOU BOUETOU,
    "Efficient movie recommendation based on the user sequential preferences and sequential profiles using HMMs",
	(Under review).
